Change Log

Aerie documentation changes are tracked using the Git configuration management tool. The code used to generate the documentation is stored in a publicly accessible GitHub repository. Incremental changes to the documentation can be viewed on a graphical timeline on GitHub.

Controlling Documents

Aerie development through AMMOS is controlled by a number documents, which are listed on a wiki page within the MGSS space on JPL's Confluence instance.
